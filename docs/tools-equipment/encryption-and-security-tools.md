## Encryption & Security Tools

| Tool | Type | Price | Tiers | Notes |
|---|---|---|---|---|
| **VeraCrypt** | Full-disk / container encryption | Free (open source) | 1–5 | Create encrypted containers for oral history files. Cross-platform. Essential for USB drives and portable storage. |
| **BitLocker** | Full-disk encryption (Windows) | Included in Windows Pro/Enterprise | 1–5 | Encrypt entire drives. Ensure it's enabled on any laptop with oral history data. Required for all tiers. |
| **FileVault** | Full-disk encryption (macOS) | Included in macOS | 1–5 | Encrypt entire drives. Enable in System Preferences → Security & Privacy. Required for all tiers. |
| **7-Zip** | File-level encryption | Free (open source) | 1–5 | AES-256 encrypted archives. Good for sharing individual files via secure transfer. |
| **Cryptomator** | Cloud encryption overlay | Free (open source) | 1–5 | Encrypts files before they go to cloud storage (OneDrive, Dropbox, etc.). Elevates cloud storage security — enables Tier 3–4 data on cloud services. |
| **KeePass / KeePassXC** | Password manager (local) | Free (open source) | 1–5 | Local database, no cloud. Preferred for Tiers 4–5 where cloud-synced passwords are undesirable. |
| **Bitwarden** | Password manager (cloud) | Free tier / $10/yr premium | 1–3 | Cloud-synced. Open source. EU/US servers. Good for team password sharing. For Tiers 4–5: prefer KeePass (local). |

**Minimum requirements for all tiers:**
- Full-disk encryption on all devices (BitLocker/FileVault)
- Encrypted USB drives for transport (VeraCrypt)
- Strong, unique passwords for all project accounts (password manager)

**Additional for Tiers 4–5:**
- File-level encryption (VeraCrypt containers or Cryptomator)
- Restricted access logging
- Physical security measures (locked cabinet for consent forms, restricted office access)
