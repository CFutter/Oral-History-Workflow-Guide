## Transcription Tools

| Tool | Type | Price | Data location | Suitable tiers |
|---|---|---|---|---|
| **OpenAI Whisper (local)** | AI transcription (runs on your machine) | Free (open source) | **Local only** — data never leaves your computer | All tiers (1–5) |
| **Whisper.cpp** | Optimized local Whisper implementation | Free (open source) | Local only | All tiers (1–5) |
| **macOS Dictation / Windows Speech** | Built-in OS transcription | Free | Local (mostly) — verify current policies | Tiers 1–3 |
| **oTranscribe** | Manual transcription web tool | Free (open source) | Local (runs in browser, no upload) | All tiers (1–5) |
| **f4transkript / f5transkript** | Manual transcription software | ~€58 (student) / ~€148 | Local | All tiers (1–5) |
| **Trint** | AI transcription (cloud) | ~$52/mo | UK/EU cloud | Tiers 1–2 (with DPA) |
| **Otter.ai** | AI transcription (cloud) | Free tier / $10–20/mo | US cloud | Tier 1 only (with DPA and caution) |
| **Amberscript** | AI + human transcription | From €8/hr (AI) / €1.50/min (human) | Netherlands (EU) | Tiers 1–3 (with DPA) |
| **MAXQDA (Transcription module)** | QDA software with transcription | University license ~€100/yr | Local | All tiers (1–5) |

**Recommended approach:**
- **Tiers 1–3:** Use local Whisper for first pass, then manually correct. Or use Amberscript with DPA.
- **Tiers 4–5:** Use local Whisper or oTranscribe/f4transkript only. No cloud processing.
