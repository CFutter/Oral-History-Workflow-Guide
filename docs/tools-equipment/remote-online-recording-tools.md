## Remote / Online Recording Tools

| Tool | Type | Price | Tiers | Notes |
|---|---|---|---|---|
| **Zoom (video conferencing)** | Video call with local recording | UZH license (free for staff) | 1–3 | Use "local recording" (not cloud). Record separate audio tracks if possible. For Tiers 4–5: use only as communication channel, record locally via OBS. |
| **Riverside.fm** | Remote recording platform | Free tier (limited) / ~$15/mo | 1–3 | Records locally on each participant's device → higher quality than Zoom. Separate audio + video tracks. Cloud upload involved — requires DPA. |
| **Zencastr** | Remote audio recording | Free tier / ~$20/mo | 1–3 | Local recording, separate tracks. Audio-focused. Cloud component — requires DPA. |
| **OBS Studio** | Screen/video recording (local) | Free (open source) | 1–5 | Fully local recording of your own screen/camera. Can capture Zoom calls as backup. No data leaves device. |
| **Cleanfeed** | Remote audio studio | Free tier / £22/mo | 1–2 | Professional remote audio recording. Low latency. No video. UK-based — check DPA. |

**Recommended for UZH:** Zoom (institutional license) with local recording enabled as baseline. For higher quality: Riverside.fm or Zencastr for the actual recording, with Zoom as the communication channel.

!!! warning "Remote recording and data protection"
    Cloud-based recording tools (Riverside, Zencastr) process personal data on external servers. Verify the service's data processing location, review their DPA, and apply UZH third-party processing requirements (see §2.8). For Tiers 4–5, consider local-only recording: each participant records locally using OBS or a phone recorder, then transfers files via secure channel.
