## Swiss University Storage Solutions

| Service | Provider | Tiers | Notes |
|---|---|---|---|
| **UZH Research Storage** | UZH Central IT | 1–5 | Primary storage for active research data. Backed up. Access controlled. Recommended for all tiers. |
| **UZH ScienceCloud** | UZH S3IT | 1–5 | Cloud computing and storage for research. Swiss-hosted. Suitable for processing sensitive data including Tier 5. |
| **UZH OneDrive / SharePoint** | Microsoft 365 (UZH license) | 1–3 | Swiss/EU data centers. Good for collaboration. For Tiers 4–5: data residency and access logging may not meet requirements — use UZH Research Storage. |
| **SWITCHdrive** | SWITCH (Swiss university network) | 1–3 | Swiss-hosted cloud storage. 50 GB free for Swiss university members. Good for sharing with collaborators at other Swiss universities. Not designed for highly restricted access. |
| **SWITCHfilesender** | SWITCH | 1–4 | Temporary large file transfer (up to 250 GB). Swiss-hosted. Temporary links — not for long-term storage. |
| **SWITCHengines** | SWITCH | 1–4 | OpenStack cloud infrastructure. Swiss data centers. For compute and storage. Requires technical setup. |

**Recommendation:** Use UZH Research Storage for primary data. SWITCHdrive for collaboration. VeraCrypt-encrypted containers for portable copies.
